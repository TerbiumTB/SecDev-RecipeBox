# P11 - DAST (OWASP ZAP Baseline) Evidence

Этот каталог содержит артефакты, автоматически сгенерированные workflow `.github/workflows/ci-p11-dast.yml`.

## Структура

- `zap_baseline.html` — человекочитаемый HTML-отчёт с результатами сканирования
- `zap_baseline.json` — JSON-отчёт с детальной информацией о найденных алертах

## Генерация артефактов

Артефакты автоматически генерируются при:
- Push в ветки с изменениями в `app/**`, `src/**` или самом workflow
- Ручном запуске workflow через `workflow_dispatch`

## Использование артефактов

1. **Просмотр артефактов**: После успешного выполнения workflow артефакты доступны в разделе "Artifacts" соответствующего workflow run
2. **Локальная генерация**: Для локальной генерации используйте команду `make dast-local` или см. документацию по OWASP ZAP
3. **Интеграция в отчёты**: Артефакты используются для формирования раздела DAST итогового отчёта

## Инструменты

- **OWASP ZAP** (baseline scan): Автоматизированное DAST-сканирование веб-приложения

## Связь с коммитами

Каждый набор артефактов привязан к конкретному коммиту и workflow run. Файлы автоматически коммитятся в репозиторий после успешного выполнения workflow.

## Интерпретация результатов

Результаты сканирования классифицируются по уровням риска:
- **High**: Критические уязвимости, требующие немедленного исправления
- **Medium**: Средние риски, рекомендуется исправить
- **Low**: Низкие риски или информационные сообщения

Найденные алерты должны быть обработаны:
- Исправление реальных уязвимостей
- Оформление waivers в `policy/waivers.yml` с обоснованием для ложных срабатываний
- Документирование принятых рисков
