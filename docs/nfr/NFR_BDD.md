**Feature:** Производительность API
> **Scenario:** p95 времени ответа при нагрузке
>    - **Given** сервис развернут
>    - **When** проводится нагрузочный тест 20 RPS в течение 5 минут
>    - **Then** p95 времени ответа не превышает 300 мс

**Feature:** Единый формат ошибок
> **Scenario:** Любое исключение оборачивается в ApiError
>    - **Given** пользователь делает запрос с невалидным телом
>    - **When** API вызывает валидационную ошибку
>    - **Then** ответ содержит поле "error" с кодом "validation_error"

**Feature:** Проверка ограничения размера запроса
> **Scenario:** Отправка слишком большого JSON
>    - **Given** клиент отправляет POST /recipes с телом > 1MB
>    - **When** запрос обработан FastAPI
>    - **Then** возвращается код 413 и тело с ошибкой "payload_too_large"

**Feature:** Доступность документации
>  **Scenario:** Проверка Swagger UI
>    - **Given** сервис запущен
>    - **When** пользователь открывает /docs
>    - **Then** отображается корректная спецификация OpenAPI без ошибок

**Feature:** Надёжность при сбое репозитория
> **Scenario:** Ошибка в хранилище не рушит приложение
>    - **Given** метод RecipeRepo выбрасывает RuntimeError
>    - **When** выполняется POST /recipes
>    - **Then** API возвращает 500 и error.code = "internal_error"
